# TamaAnkenka

## Tóm tắt
- Mục đích: [Xử lý Ankenka cho Tama].

## Flow (quy trình)
1. Nhận input.
2. Chạy `tasks.py` để xử lý.
3. Có thể dùng `案件化.xlsm` hoặc `タマホーム_案件化+資料UP_V1_2.exe` trong flow.
4. Lưu/Upload kết quả.

## Các file cần thiết
- `tasks.py`
- `案件化.xlsm` — mẫu Excel
- `タマホーム_案件化+資料UP_V1_2.exe` — helper exe (nếu cần)

---

## Chạy & Debug
- Kiểm tra file Excel mẫu trước khi chạy; `exe` có thể cần quyền Windows để chạy.
