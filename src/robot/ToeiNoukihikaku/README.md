# ToeiNoukihikaku

## Tóm tắt
- Mục đích: [Xử lý Toei Noukihikaku, tạo/gửi kết quả].

## Flow (quy trình)
1. Nhận input.
2. Xử lý bằng `tasks.py` và có thể sử dụng `touei_noukihikaku_V1_4.exe`.
3. Sinh `結果.xlsx` và upload.

## Các file cần thiết
- `tasks.py`
- `touei_noukihikaku_V1_4.exe`
- `結果.xlsx` — mẫu/đầu ra

## Chạy & Debug
- Kiểm tra script/exe bằng môi trường Windows khi cần.
