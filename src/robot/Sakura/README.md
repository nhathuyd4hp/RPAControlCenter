# Sakura

## Tóm tắt
- Mục đích: [Xử lý task liên quan Sakura].

## Flow (quy trình)
1. Nhập input hoặc tải từ nguồn.
2. Dùng `automation/` để lấy hoặc thao tác trên site.
3. Lưu file tạm trong `downloads/` và xử lý.
4. Xuất kết quả và upload.

## Các file cần thiết
- `tasks.py`
- `automation/`
- `downloads/` — nơi lưu tệp tạm

---

## Chạy & Debug
- Kiểm tra `downloads/` để debug các file input trước khi xử lý.
